#### TC-E2E-EDIT-001：display_name 内联编辑

**目的**：验证点击名称后进入编辑模式，Enter 键保存，值在刷新后持久化。

**测试步骤**：
```typescript
test('display_name 内联编辑', async ({ page }) => {
  const dashboard = new DashboardPage(page);
  await dashboard.goto();

  await dashboard.editDisplayName('%1', '重构认证模块');

  // 验证 UI 立即更新
  const card = await dashboard.getCardByPaneId('%1');
  await expect(card.locator('[data-testid="display-name"]')).toContainText('重构认证模块');

  // 刷新页面后验证持久化
  await page.reload();
  await dashboard.goto();
  const cardAfterReload = await dashboard.getCardByPaneId('%1');
  await expect(cardAfterReload.locator('[data-testid="display-name"]')).toContainText('重构认证模块');
});
```

**预期结果**：
- 点击名称后出现 input 框，原文本作为初始值
- 输入新名称后按 Enter，input 框消失，显示新名称
- 页面刷新后名称依然为 `"重构认证模块"`

**失败排查**：
- 若刷新后名称丢失：检查 PATCH 请求是否发出（Network 面板）；检查服务端是否正确执行了 `await db.commit()`
- 若 input 框不出现：检查 `[data-testid="display-name"]` 元素是否绑定了 `onClick` 事件

#### TC-E2E-EDIT-002：Escape 键取消编辑

**目的**：验证按 Escape 键取消编辑，恢复原始值，不触发 API 请求。

**测试步骤**：
```typescript
test('Escape 取消编辑', async ({ page }) => {
  const card = await dashboard.getCardByPaneId('%1');
  const originalName = await card.locator('[data-testid="display-name"]').textContent();

  await card.locator('[data-testid="display-name"]').click();
  await page.keyboard.type('临时名称不应保存');
  await page.keyboard.press('Escape');

  await expect(card.locator('[data-testid="display-name"]')).toContainText(originalName!);
});
```

**预期结果**：
- 按 Escape 后 input 框消失
- 显示原始名称，未发生变化
- 无 PATCH 请求被发出（可通过 `page.waitForRequest` 的 timeout 验证）

#### TC-E2E-EDIT-003：Tag 添加与删除

**目的**：验证点击 [+] 添加 tag，点击 tag 本身删除。

**测试步骤**：

## 4.6 快捷回复与输入发送

> **注意**：所有实际发送测试默认使用 dry-run 模式（`confirm: false`），防止测试指令污染真实 tmux 环境。只有专门标记 `@realSend` 的用例会执行真实发送，且只对隔离的测试 pane 执行。

#### TC-E2E-INPUT-001：快捷回复按钮——Yes

**目的**：验证点击 [✓ Yes] 按钮后出现确认对话框，Cmd+Enter 确认后完成发送流程。

**前置条件**：pane `%1` 处于 `waiting_input` 状态，agent 输出 `"Continue? [y/n]"`。

**测试步骤**：
```typescript
test('快捷回复 Yes', async ({ page }) => {
  const card = await dashboard.getCardByPaneId('%1');

  await expect(card.locator('[data-testid="quick-reply-yes"]')).toBeVisible();
  await card.locator('[data-testid="quick-reply-yes"]').click();

  // 确认对话框出现，包含目标信息和预览
  await expect(page.locator('[data-testid="send-confirm-dialog"]')).toBeVisible();
  await expect(page.locator('[data-testid="confirm-preview"]')).toContainText('y');
  await expect(page.locator('[data-testid="confirm-target"]')).toContainText('%1');

  // Cmd+Enter 确认
  await page.keyboard.press('Meta+Enter');

  await expect(page.locator('[data-testid="send-confirm-dialog"]')).not.toBeVisible();
  await expect(card.locator('[data-testid="sent-echo"]')).toContainText('已发送: y');
});
```

**预期结果**：
- 点击后弹出确认对话框，显示目标 pane 和发送内容预览
- Cmd+Enter 后对话框消失，卡片显示发送回显（10 秒后消失）

**失败排查**：
- 若确认对话框未出现：检查 `QuickReply.tsx` 中点击事件是否正确触发了 `showConfirmDialog` 状态
- 若 `confirm-target` 未显示 pane_id：检查确认对话框组件是否接收并展示了 `paneId` prop

#### TC-E2E-INPUT-003：自定义输入框——普通 Enter 不发送

**目的**：验证自定义输入框中普通 Enter 不触发发送，只有 Cmd+Enter 才触发。

**测试步骤**：
```typescript
test('普通 Enter 不触发发送', async ({ page }) => {
  const card = await dashboard.getCardByPaneId('%1');
  const input = card.locator('[data-testid="custom-input"]');

  await input.fill('请继续，忽略警告');

  // 普通 Enter 不触发确认对话框
  await page.keyboard.press('Enter');
  await expect(page.locator('[data-testid="send-confirm-dialog"]')).not.toBeVisible();

  // Cmd+Enter 触发
  await page.keyboard.press('Meta+Enter');
  await expect(page.locator('[data-testid="send-confirm-dialog"]')).toBeVisible();
  await expect(page.locator('[data-testid="confirm-preview"]')).toContainText('请继续，忽略警告');
});
```

## 4.7 跳转 iTerm2 功能

**文件**：`tests/e2e/test_focus.spec.ts`

> **注意**：此测试需要 iTerm2 已安装且已授权辅助功能权限。CI 环境通过 `ATWA_SKIP_ITERM2=1` 跳过实际 AppleScript 调用，只验证 API 调用链。

---

#### TC-E2E-FOCUS-001：点击 Focus 按钮触发 API 调用

**测试步骤**：
```typescript
test('Focus 按钮触发 API', async ({ page }) => {
  const focusRequest = page.waitForRequest(
    req => req.url().includes('/api/sessions/%1/focus') && req.method() === 'POST'
  );

  const card = await dashboard.getCardByPaneId('%1');
  await card.locator('[data-testid="focus-btn"]').click();

  const req = await focusRequest;
  expect(req).toBeTruthy();
});
```

**预期结果**：
- 点击后发出 `POST /api/sessions/%1/focus` 请求
- 请求在 1 秒内发出
